# Альфа-Банк ML Hackathon

Система для предсказания дохода клиентов на основе ML модели.

## Архитектура

- **Backend**: FastAPI + SQLAlchemy + CatBoost
- **Frontend**: React + TypeScript + Material-UI
- **Database**: PostgreSQL
- **Web Server**: Nginx
